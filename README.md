# Taco Cloud (Spring Boot Application)

A web application for designing and ordering custom tacos, built while following ["Spring in Action"](https://www.manning.com/books/spring-in-action-fifth-edition) by [Craig Walls](https://en.wikipedia.org/wiki/Craig_Walls).

## Tech Stack
-Java
-Spring Boot
-Spring MVC
-Thymeleaf
-Hibernate
-JPA
-Spring Security

## Features
- Design custom tacos with multiple ingredients
- Persist taco orders using database integration
- User authentication and authorization (Spring Security)
- Dynamic UI using Thymeleaf templates

## How to Run

1. Clone the repository
2. Open in IntelliJ / Eclipse
3. Run the main Spring Boot application class using mvn spring-boot:run command in the project repository
4. Visit: http://localhost:8080
